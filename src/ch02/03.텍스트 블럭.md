# 03. 텍스트 블럭

JDK 15부터 다중행 문자열 리터럴을 작성할 수 있는 기능을 제공하며 이를 `텍스트 블럭`이라 한다.

```java
String str = "class Main {\n"
    + "\tpublic String main(String[] args) {\n"
    + "\t\tSystem.out.println(\"hello world\");\n"
    + "\t}\n"
    + "}\n";
```

기존 방식에서는 위처럼 여러 특수문자를 섞어서 문자열을 작성해야 했다. 아래는 텍스트 블럭을 활용한 방식이다.

```java
String str = """
        class Main {
            public String  main(String[] args) {
                System.out.println("hello world");
            }
        }
        """;
```

압도적으로 간편하고 직관적으로 보인다. 다만 아래 내용을 주의하도록 한다.

⭐ 닫는 ```"""```의 위치에 의해 들여쓰기의 기준이 된다.
