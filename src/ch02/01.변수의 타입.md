# 01. 변수의 타입

변수를 선언할 때는 저장하려는 값의 특성을 고려하여 가장 알맞은 자료형을 선택한다.<br>
이때 자료형은 크게 다음과 같이 분류할 수 있는데

- 기본형(primitive type) : ```boolean```, ```char```, ```byte```, ```short```, ```int```, ```long```, ```float```, ```double``` 
- 참조형(reference type) : 객체의 주소를 저장, 기본형을 제외한 나머지 타입

기본형 변수는 실제 값을 가지며, 참조형 변수는 어떤 값이 저장되어 있는 주소를 가진다.<br>
참조형 변수는 클래스의 이름이 참조 변수의 타입이 된다.


⭐ 변수의 타입이 기본형이 아닌 것들은 모두 참조변수

---

참조형 변수는 ```null``` 또는 객체의 주소를 값으로 갖는다.<br>

```java
Date today = new Date();
```

객체를 생성하는 연산자 new의 결과는 생성된 객체의 주소이다.<br>
따라서 today는 생성된 Date() 객체의 주소값이 담기게 된다.

⭐ JVM이 32bit라면 주소값의 크기는 4Byte이며, 64bit라면 주소값의 크기는 8Byte이다.

---

정수 자료형의 경우 다음과 같은 범위를 가진다.

```math
-2^(n-1) ~ 2^(n-1) - 1 , n은 비트 수
```

즉 4Byte에 해당하는 ```int```의 경우, ```-2^31 ~ 2^31 - 1```의 범위<br>

⭐ 2^10 = 1024 ~= 10^3 이기에, 2^31 = 2^10 * 2^10 * 2^10 * 2 ~= 2*10^9 (약 20억)의 최댓값을 가진다.
