# 02. String클래스 주요 메서드

- char charAt(int index)
- int length()
- String substring(int from, int to)
- boolean equals(Object obj)
- char[] toCharArray()

```java
String str = new String("hello world");
String str2 = new String("helloworld");

System.out.println(str.charAt(7)); // o
System.out.println(str.length()); // 11
System.out.println(str.substring(1, 4)); // ell
System.out.println(str.equals(str2)); // false
System.out.println(str.toCharArray()); // hello world
```

`System.out.println(str.toCharArray());`의 경우 "hello world" 문자열을 `char[]`로 변환한 후,<br>
`print`하였기에 `char[]` 배열의 예외적 사항으로 인해 "hello world" 문자열이 출력된다.

---

`char[]` 배열과 `String` 문자열간의 변환은 아래와 같다.

- char[] -> String

```java
char[] chArr = new char[]{'a', 'p', 'p', 'l', 'e'};

String s = new String(chArr); // "apple"
```

- String -> char[]

```java
String s = "apple";

char[] chArr = s.toCharArray(); // ['a', 'p', 'p', 'l', 'e']
```
