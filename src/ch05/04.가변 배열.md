# 04. 가변 배열

가변 배열을 필요로 할 경우 다음과 같이 생성할 수 있다.

```java
int[][] arr = new int[5][];

arr[0] = new int[10];
arr[1] = new int[2];
arr[2] = new int[0];
arr[3] = new int[5];
arr[4] = new int[7];
```

2차원 이상의 다차원 배열을 생성할 때는 전체 배열 차수 중 마지막 차수의 길이를 지정하지 않고<br>
추후에 각기 다른 길이의 배열을 생성함으로써 고정된 형태가 아닌

보다 유동적인 가변 배열을 구성할 수 있다.
